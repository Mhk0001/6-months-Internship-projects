# 🛍️ Super Mall Web Application

## 📌 Overview

The **Super Mall Web Application** is a modern e-commerce platform that allows users to explore shops, view products, compare offers, and check store locations. It also includes an admin panel to manage products and offers efficiently.

---

## 🚀 Features

### 👤 User Features

* Browse products from different shops
* View product details with images and pricing
* Compare offers and discounts
* Explore shop categories (Electronics, Clothing, Books, etc.)
* View shop locations

---

### 🔐 Admin Features

* Secure admin login
* Add new products
* Edit existing products
* Delete products
* Manage offers and discounts

---

## 🏬 Shop Categories

* Electronics Store
* Fashion Store
* Book Store
* Classic Store

---

## 💻 Tech Stack

* Frontend: React.js
* Backend: Firebase (Authentication + Firestore)
* Styling: CSS / Bootstrap

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/repo-name.git
```

### 2️⃣ Navigate to project folder

```bash
cd SuperMall
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Setup environment variables

Create a `.env` file and add:

```env
REACT_APP_FIREBASE_API_KEY=your_api_key
REACT_APP_FIREBASE_AUTH_DOMAIN=your_domain
REACT_APP_FIREBASE_PROJECT_ID=your_project_id
REACT_APP_FIREBASE_STORAGE_BUCKET=your_bucket
REACT_APP_FIREBASE_MESSAGING_SENDER_ID=your_id
REACT_APP_FIREBASE_APP_ID=your_app_id
```

---

### 5️⃣ Run the project

```bash
npm start
```

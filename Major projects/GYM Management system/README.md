# Gym Management System

This is a web-based Gym Management System built using HTML, CSS, and JavaScript (localStorage-based).

## Features

- User Registration & Login (localStorage)
- Member Management
- Bill Generation
- Diet Plan Display
- Supplement Store Page
- Contact Form
- Protected Dashboard (after login only)

## Technologies Used

- HTML5
- CSS3
- JavaScript (vanilla)
- Browser LocalStorage (for data persistence)

## How to Use

1. Open `index.html` to visit landing page.
2. Sign up via `signup.html`.
3. Login via `login.html`.
4. Navigate using dashboard.

> No backend or server required. Runs fully in browser.

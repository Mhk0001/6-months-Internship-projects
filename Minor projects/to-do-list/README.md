# 📝 My To-Do List App

My To-Do List App is a simple and interactive web application that helps users manage their daily tasks efficiently. Users can add, edit, mark tasks as completed, and remove tasks easily through a clean and user-friendly interface.

This project is built using **HTML, CSS, and JavaScript** and uses **Webpack** for bundling and development.

---

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as completed
- 🗑️ Delete individual tasks
- 🧹 Clear all completed tasks
- 🎨 Clean and responsive UI
- 💾 Tasks persist using browser local storage

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Webpack  
- npm
